OWASP Web Security Testing Guide

I’m an author and maintainer of the Open Web Application Security Project (OWASP) Web Security Testing Guide (WSTG), version 5. The WSTG is a comprehensive guide to testing the security of web applications and web services, and relied upon by organizations and penetration testers worldwide.
