About Victoria Drake

Hello! I’m Victoria Drake, and I’m a software security developer. You’ve found my blog, where I write about how to design and build secure software applications that are inherently fortified against attack. I also geek out over computer science topics, command line tools, and good programming practices.

I’m an author and maintainer for the nonprofit Open Web Application Security Project’s Web Security Testing Guide. It’s the foremost open source resource for organizations and developers worldwide to learn how to test web application security. I write content, edit contributions, and influence project direction.

I’m passionate about building tools to help people save time and do more with their lives. Among others, I developed ApplyByAPI.com. It’s an application filter that helps you hire developers by only letting through candidates you may actually want to interview. You can use it to find quality developers while reducing time spent on hiring by up to 79%.

I also write technical articles and tutorials for a number of publications. You can find me in The Startup, Better Programming, Towards Data Science, Hacker Noon, and more. I’m a recognized Distinguished Author on dev.to, and a twice-awarded top annual contributor to freeCodeCamp, a 501(c)(3) non-profit organization that helps millions of people worldwide learn how to code.

For fun, I design and code static site generator themes like this one for Hugo and a super-nostalgic Visual Studio Code theme named Kabukichō.

I spent a few years traveling the world with just one bag, and now spend most of my time around Washington, DC. When I’m not on a computer, I’m probably making pour-over coffee or practicing my zombie apocalypse skills. To ask me anything, say hello@victoria.dev.
